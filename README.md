# MSX ROM Mapper Identification Tool

MSX実機において、指定したスロットのROMマッパーを判定するツールです。
<br>MSXゲームカートリッジアダプタ(仮称・以下アダプタ)におけるカートリッジ認識アルゴリズムの試作になります。

動作報告、不具合報告は歓迎いたしますが、前述の通りなので…
- 動作にはカートリッジの後挿しか切り離しスイッチ付きのスロットが必要です(後挿しは自己責任で！)
- 複数回の判定で結果が違っても初回が正常判定なら問題ありません
- 他のマッパーとして動作するもの(R-TYPE・スーパーピエロ等)は動作するマッパーとして判定します
- マッパーを操作して判定できるもの限定です(D/Aコンバータ・PCM再生ICなどの搭載はROMイメージ側での判定が必要)

クロスブレイム・スーパーロードランナー・は～りぃふぉっくす雪の魔王編などのように特別に対応が必要なものも、要望が多ければできる限り対応する予定ですので、ご要望をお待ちしております。アダプタでの対応については(判定だけでなくマッパーの実装も必要なため)MSXアソシエーション宛に要望をお送りください。
<br>同人ベースなど小ロット生産の「ウチのカートリッジ」のマッパーにも対応してくれ、などの要望も歓迎です(対応できるとは限りませんが…)。ちなみにウチの似非²RAMは普通に似非RAM/MegaSRAMとして認識します。

ただし海外の「N in 1」系のマッパーは今後も対応の予定はありません。

# 判定可能なマッパーについて

以下のマッパーが判定可能です(そのはずです(汗))。
<br>番号はアダプタのための便宜上のもので、それ以外に意味はありません。欠番は公式エミュ内部で使用している番号ですが判定からは外しています。

|番号|名称
|:--|:--|
|0	|マッパーなし(通常ROM)|
|1	|GENERIC 8K|
|2	|GENERIC 16K|
|3	|アスキー 8K|
|4	|アスキー 16K|
|5	|コナミ非SCC(新10倍カートリッジバックアップRAM判定対応)|
|6	|SCC|
|7	|SCCサウンドカートリッジ(RAM容量判定対応)|
|8	|PAC/FMPAC(バックアップRAMのみROMマッパー非対応)|
|10	|クロスブレイム|
|12	|似非RAM/MegaSRAM 8K|
|13	|似非RAM/MegaSRAM 16K|
|14	|はぁりぃふぉっくす雪の魔王編|
|15	|スーパーロードランナー|
